# Syncro3
Syncro3 is a hardware groovebox merging a poly-synth, drum machine, and 16-step sequencer. It challenges a 3-person team to sync melody, rhythm, and logic in real-time. By managing shared CPU resources and low-latency DSP, it delivers a unified, professional multitimbral instrument.

## Run Instructions

Requirements:
- Java installed and available on your PATH

From the project root, run:

```bat
javac DAW\src\*.java
java -cp DAW\src SequencerTest
```

On Windows, you can also double-click `run.bat` in the repository root.
